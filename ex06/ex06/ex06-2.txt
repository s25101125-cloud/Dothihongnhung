Adding a tiling background image

1. Open the file bistro.html.

2. Add a declaration to the body style rule that makes the image bullseye.png tile in the background of the page. Be sure to include the pathname relative to the style sheet.
	background-image: url(images/bullseye.png);

3. In the #header rule, add the image "purpledot.gif" and set it to repeat horizontally only.
	#header {
		margin-top: 0;
		padding: 3em 1em 2em 1em;
		text-align: center;
		background-color: rgba(255,255,255,.5);
		background-image: url(images/purpledot.png);
		background-repeat: repeat-x;
	}
